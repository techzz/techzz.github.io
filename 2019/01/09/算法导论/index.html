渐进符号

O：<=, n^2+O(n)=O(n^3)
Ω：>=, √n=Ω(lgn)
ω: > , 
o: < , 2n^2=o(n^3)
Θ: = : n^2=Θ(n^2)

递归复杂度

  1. 代换法（数学归纳法）
  2. 递归树
  3. 主方法（有限制）：递归方程形如 T(n)=aT(n/b)+f(n) a>=1,b>1
    • f(n)=O(n^(logba-ε)) T(n)=Θ(n^logba)
    • f(n)=Θ(n^logba*(lgn)^k) T(n)=Θ(n^logba*(lgn)^(k+1))
    • f(n)=Ω(n^(logba+ε)) && af(n/b)<=(1-ε’)f(n) T(n)=Θ(f(n))
      (0<ε<logba,,0<ε’<1,k>=0 常数)

分治法

  1. 归并排序
  2. 二分查找
  3. 乘方定理
    1. 普通相乘 T(n)=Θ(n)
    2. 分而治之
      • n=even,x^n=x^(n/2)*x^(n/2)
      • n=odd,x^n=x^(n-1/2)*x^(n-1/2)*x
      • T(n)=Θ(lgn)
  4. 斐波那契数列
    1. 普通递归 T(n)=Θ(φ^n) φ为黄金分割比
    2. 从F1到Fn T(n)=Θ(n)
    3. Fn=φ^n/√5 T(n)=Θ(lgn) (会发生精度丢失)
    4. 2×2矩阵((Fn+1),Fn),(Fn,Fn-1))=((1,1),(1,0))^n T(n)=Θ(lgn)
      3,4运用乘方定理
  5. 矩阵乘法（C=A×B）
    1. 普通运算 T(n)=Θ(n^3)
    2. n×n矩阵转化成 4个 n/2×n/2 矩阵
      T(n)=8T(n/2)+Θ(n^2)==>T(n)=Θ(n^3)
    3. Strassen’s Algorithm
      T(n)=7T(n/2)+Θ(n^2)==>T(n)=Θ(n^2.81)

  6. VLSI

    1. 二叉树画法
      H(n)=H(n/2)+Θ(1)==>H(n)=Θ(lgn)
      W(n)=2W(n/2)+Θ(1)==>W(n)=Θ(n)
      T(n)=H(n)*W(n)=Θ(nlgn)

    2. H型画法

      H(n)=2H(n/4)+Θ(1)==>H(n)=Θ(√n)
      W(n)=2W(n/4)+Θ(1)==>W(n)=Θ(√n)
      T(n)=H(n)*W(n)=Θ(n)

分治法也是一种递归，可通过递归复杂度的计算方法构造不同的分治思路

排序

  1. 基于比较排序模型（决策树）
    1. 冒泡排序 O(n^2) 稳定 O(1)
    2. 插入排序 O(n^2) 稳定 O(1)
    3. 选择排序 O(n^2) 不稳定 O(1)
    4. 快速排序 O(nlgn) 不稳定 O(lgn)
      • 最坏O(n^2)
      • 平均O(nlgn) <– 随机化算法
    5. 希尔排序 O(nlgn) 不稳定 O(1)
    6. 堆排序 O(nlgn) 不稳定 O(1)
    7. 归并排序 O(nlgn) 稳定 O(n)
  2. 非比较排序模型
    1. 计数排序 O(n+k) 稳定 O(k)
      • k为n个待排序数字的最大值，当k不是很大时适用
    2. 桶排序
    3. 基数排序 O(d(n+r)) 稳定 O(n+2*r)
      • d为最大位数，r为每一位位数的最大值
